# Python-Simple-Video-Editor
 - Very Simple Video Editor, Fully Modular (you can make your own effects)

Run the Install Requirements file



Audio Manager
 - This is for removing audio to edit it alone in audacity or something else



They all work the same way
 - Put your Videos in the VIDEOS folder
 - Run the Start file
 - After export, your videos will be in the EXPORT folder
 - (Combine script export to COMBINED, finds videos from EXPORT)



The Effect Editor in the "Editor" folder is a fully modular editor
In the EFFECTS folder, you can create your own effect scripts, and the main will automatically see it
